Rules
  • Buttons: translateY(0 → -1px) + teal glow on hover, 120ms.
  • Cards: border colour --border-2 → --teal, 200ms. No translate.
  • Reveal on scroll: opacity 0 → 1 + translateY(12px → 0), 320ms, stagger 40ms.
  • All colour transitions use cubic-bezier(0.2, 0, 0, 1).
