# Голос и тон LeadUp AI

## Наш голос

LeadUp AI говорит как **умный коллега, который уже прошёл этот путь**:
- Практично, без воды
- Уважительно, без снисхождения
- Современно, без жаргона ради жаргона
- На «ты» (если не корпоративный клиент)

## Что мы делаем

| Делаем | Не делаем |
|--------|-----------|
| Объясняем через примеры | Теоретизируем в вакууме |
| Признаём ограничения AI | Обещаем волшебные результаты |
| Говорим о реальных кейсах | Используем buzzwords без объяснения |
| Используем юмор ситуативно | Шутим там, где это неуместно |
| Пишем кратко | Растягиваем на несколько абзацев то, что влезает в одно |

## Тон по каналам

| Канал | Тон |
|-------|-----|
| Telegram (Нейросборка) | Дружелюбный, живой, чуть неформальный |
| YouTube/Reels | Энергичный, структурированный, с «хуком» |
| LinkedIn | Экспертный, деловой, с данными |
| Email-рассылка | Личный, прямой, с одним CTA |
| B2B-презентация | Профессиональный, ориентированный на ROI |

## Запрещённые слова

- «революционный», «прорывной», «уникальный» (если нет доказательств)
- «просто», «легко» (обесценивает усилия клиента)
- «AI заменит» (страх не конвертит, смысл конвертит)
